# Huzaira Sultan — Portfolio Website

Personal portfolio website built with pure HTML, CSS, 
and JavaScript — no frameworks.

## 🔗 Live Demo
https://portfolio-nine-ecru-rhkeiy61vy.vercel.app/#projects

## Built With
HTML5 · CSS3 · JavaScript · GSAP

## Features
- Custom cursor (dot + ring effect)
- Canvas-based particle background
- Scroll reveal animations
- Typing effect with rotating roles
- Animated counters
- Skill progress bars
- 3D tilt hover cards
- Magnetic buttons
- Dark/Light mode toggle
- Mobile responsive
- Scroll progress bar

## Skills Used
HTML · CSS · JavaScript · GSAP · Responsive Design · 
UI/UX · CSS Animations · Web Development
